# 프로젝트명 : 가위바위보 게임과 로봇팔 통신<br/>

<hr/>

1. 기간 : 2023.04.24 ~ 2023.04.28<br/>
2. 인원 : 1명<br/>

<hr/>

## 목표
1. 부채질 하는 로봇팔 만들기 <br/>
2. 가위바위보 게임 만들기 <br/>
3. 로봇팔과 게임을 통신하여 게임에서 지면 로봇팔이 움직여서 때린다.<br/>
   로봇팔 깃허브 : (https://github.com/pflnhw/robot_1)
   
<hr/>

* Arduino
* C++
* Github
